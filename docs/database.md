# Database

The **database** of the Online Gradebook is based on **SQL** and stores all data used by the application.  
It contains information about users, students, teachers, subjects, grades, and absences.  
Relationships between tables ensure that each student, teacher, and subject are properly connected, allowing the system to function consistently.  
The database serves as the foundation that links the backend logic with the data shown on the frontend.

---

**Baza podataka** Online Dnevnika zasniva se na **SQL** tehnologiji i čuva sve podatke koje aplikacija koristi.  
Sadrži informacije o korisnicima, učenicima, profesorima, predmetima, ocenama i izostancima.  
Veze između tabela obezbeđuju pravilno povezivanje učenika, profesora i predmeta, čime se postiže konzistentnost sistema.  
Baza podataka predstavlja osnovu koja povezuje logiku backend dela sa prikazom informacija na frontendu.
