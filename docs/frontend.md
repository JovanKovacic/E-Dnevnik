# Frontend

## English

The **frontend** of the Online Gradebook is responsible for displaying information and enabling user interaction.  
It is developed using **Vue**, providing a clear and simple web interface through which teachers and students can access the system.  
Teachers can view students, add or delete grades, and record absences, while students can log in to see their grades and attendance records.  
The frontend communicates with the backend through requests and displays data dynamically based on user actions.

## Srpski

**Frontend** Online Dnevnika zadužen je za prikaz informacija i interakciju korisnika sa sistemom.  
Razvijen je pomoću **Vue** frameworka i pruža jednostavan i pregledan web interfejs kroz koji profesori i učenici pristupaju sistemu.  
Profesori mogu da pregledaju učenike, dodaju ili brišu ocene i evidentiraju izostanke, dok učenici mogu da se prijave i vide svoje ocene i evidenciju izostanaka.  
Frontend komunicira sa backendom putem zahteva i dinamički prikazuje podatke u zavisnosti od akcija korisnika.
