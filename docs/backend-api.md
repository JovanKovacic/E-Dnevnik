# Backend

The **backend** of the Online Gradebook is built using **Java** and **Spring Boot**.  
It manages all business logic, user roles, authentication, and communication with the SQL database.  
The backend processes requests from the frontend, retrieves or modifies data in the database, and returns responses in a structured format.  
Its main goal is to ensure data consistency, reliability, and proper separation between users and their permissions.

---

**Backend** Online Dnevnika izrađen je u **Javi** koristeći **Spring Boot**.  
Upravlja poslovnom logikom, korisničkim ulogama, autentifikacijom i komunikacijom sa SQL bazom podataka.  
Backend obrađuje zahteve koji dolaze sa frontenda, pristupa bazi radi čitanja ili izmene podataka i vraća odgovore u strukturisanom formatu.  
Glavni cilj backend dela je da obezbedi tačnost podataka, pouzdan rad sistema i jasnu podelu prava između korisnika.
