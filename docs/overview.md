# Project Overview

The **Online Gradebook** application simulates a real electronic school diary.  
The goal of the project is to provide a simple and efficient way to manage student grades and absences.

The system will be used by two main groups of users: **students** and **teachers**.

- **Teachers** will have the ability to add and delete grades, as well as record student absences.  
  Additionally, they will be able to view the list of all students and their grades by subject.

- **Students** will be able to log into the system and view their grades and absences, but will not have permission to modify them.

The application will function as a **web-based system** consisting of:
- **Frontend** – developed using the Vue framework, providing the user interface,
- **Backend** – implemented in Spring Boot, handling server-side logic and communication with the database,
- **SQL database** – used for storing information about users, grades, subjects, and absences.

# Pregled projekta

Aplikacija **Online Dnevnik** predstavlja simulaciju stvarnog školskog elektronskog dnevnika.  
Cilj projekta je omogućiti jednostavan i efikasan način vođenja evidencije o ocenama i izostancima učenika.

Sistem će koristiti dve glavne grupe korisnika: **učenici** i **profesori**.

- **Profesori** će imati mogućnost dodavanja i brisanja ocena, kao i evidentiranja izostanaka za učenike.  
  Pored toga, moći će da pregledaju listu svih učenika i njihove ocene po predmetima.

- **Učenici** će imati mogućnost prijavljivanja u sistem i pregleda svojih ocena i izostanaka, ali neće moći da ih menjaju.  

Aplikacija će funkcionisati kao **web sistem** koji se sastoji iz:
- **Frontend dela** – razvijenog pomoću Vue frameworka, koji predstavlja korisnički interfejs,
- **Backend dela** – izrađenog u Spring Boot okviru, koji obezbeđuje serversku logiku i komunikaciju sa bazom podataka,
- **SQL baze** – namenjene čuvanju informacija o korisnicima, ocenama, predmetima i izostancima.
